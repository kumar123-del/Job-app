# 🧑‍💻 JobApp – Job Portal Web Application

A simple **Spring Boot + JSP + PostgreSQL** Job Portal where users can:
- View all job posts
- Add new job posts
- Apply for jobs
- View applied jobs

---

## 🚀 Tech Stack
- **Java 17+**
- **Spring Boot (MVC)**
- **PostgreSQL**
- **Hibernate (JPA)**
- **JSP + JSTL + Bootstrap**
